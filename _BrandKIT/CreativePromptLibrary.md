# XPLORATRIP — Creative Prompt Library

Plantillas de prompts para generar assets visuales premium de XPLORATRIP.

Todos los prompts respetan la paleta oficial de marca:

- Dorado: `#eabe48`
- Grafito: `#303030`
- Cian: `#00c2cb`
- Blanco: `#ffffff`

Todos los diseños son verticales 1080×1920 salvo indicación distinta.

---

## Estructura maestra de prompt

Cuando se genera un flyer, usar esta estructura base:

```
Create a premium vertical advertising flyer for XPLORATRIP in Instagram Story format,
1080x1920, with a sophisticated corporate travel aesthetic.

Use a dark graphite #303030 background with soft studio lighting, warm golden glow,
and elegant cyan #00c2cb accents. Keep the design clean, modern, high-end, and
mobile-readable.

Place the XPLORATRIP logo at the top right, small and crisp.

Main visual:
[describe the central visual]

Headline:
"[insert headline]"

Subheadline:
"[insert subheadline]"

Add supporting visual elements:
[icons, route line, map, document, airplane, family, WhatsApp icon]

CTA:
"[insert CTA]"

Design requirements:
Use modern sans-serif typography similar to Montserrat or Inter.
Make the headline large, bold, and white.
Use gold for premium accents and cyan for route/digital details.
Keep generous negative space. Avoid clutter.
Make the flyer emotional, professional, and conversion-focused.
```

## Negative Prompt Guidance

Evitar siempre en cualquier prompt visual de XPLORATRIP:

- Estilo de stock photo barato o genérico
- Estilo cartoon o caricatura
- Colores sobresaturados o estridentes
- Gráficos de emergencia o advertencia en rojo
- Sellos gubernamentales oficiales
- Documentos reales o números de identificación falsos
- Texto pequeño, ilegible o recargado
- Demasiados íconos en un mismo plano
- Composición desordenada o saturada de elementos
- Expresiones faciales de estrés, desesperación o miedo
- Texto en español con errores ortográficos o gramaticales
- Banderas de países a menos que se soliciten específicamente
- Elementos 3D excesivos o de baja calidad
- Gradientes de baja calidad o degradados abruptos
- Logo mal posicionado o sin margen adecuado
