## Reglas

- Mezclar always 1-2 base de marca + 1-2 del pilar + 1-2 geográficos relevantes.
- NO usar más de 10 hashtags por pieza (Instagram penaliza el spam de hashtags).
- NO usar hashtags de eventos no relacionados.
- NO usar hashtags genéricos masivos (#love, #travel sin contexto).
- Si la pieza nombra una ciudad específica, usar el hashtag geográfico correspondiente.
- Rotar combos para no repetir el mismo set en cada pieza.
