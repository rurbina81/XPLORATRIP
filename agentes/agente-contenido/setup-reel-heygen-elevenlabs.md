# Setup de Reel con avatar (HeyGen) + voz (ElevenLabs)

Guía concreta para producir el Reel del guion. **No renderizar en vivo** — preparar antes.

## Opción A — Solo HeyGen (más simple, recomendado para empezar)

1. Entra a HeyGen → **Create Video → Avatar**.
2. Elige tu **avatar/clon** (si ya tienes uno entrenado, úsalo; si no, un Instant Avatar o uno de stock).
3. Idioma: **Español (Latinoamérica)**.
4. Pega el **guion** (campo de texto del avatar). HeyGen genera la voz.
5. Encuadre vertical **9:16** para Reel. Activa **captions** automáticos.
6. Añade b-roll/imágenes en los cortes que marca el guion.
7. **Export** en 1080×1920. Descarga el .mp4.

## Opción B — HeyGen + ElevenLabs (voz más natural / tu clon de voz)

1. En **ElevenLabs**:
   - Elige tu **voz clonada** (Voice Lab) o una voz en español.
   - Ajustes sugeridos para hablar natural: **Stability 45-55%**, **Similarity 75-85%**, **Style 0-15%**.
   - Pega el guion → **Generate** → descarga el audio (.mp3).
2. En **HeyGen**:
   - Crea el video con el avatar y selecciona **Upload audio** → sube el .mp3 de ElevenLabs.
   - HeyGen sincroniza los labios al audio.
   - Vertical 9:16, captions, export 1080×1920.

## Tips de producción

- Frases cortas y pausas → el avatar se ve más humano.
- Primer 1 segundo = hook visual + texto en pantalla.
- Subtítulos siempre (la mayoría ve sin sonido).
- Guarda el .mp4 final como backup del vivo, en la misma carpeta donde se entrega el pack de contenido.

## Cuentas

El usuario tiene HeyGen y ElevenLabs disponibles. Pedir API key/login solo si se va a automatizar; para el webinar alcanza con la interfaz web.
