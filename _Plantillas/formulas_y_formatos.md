# XPLORATRIP — Plantillas: Fórmulas narrativas y formatos

## Fórmulas narrativas preferidas

Usar estas estructuras como base para cualquier pieza:

| Fórmula | Descripción |
|---|---|
| Problema → claridad → CTA | Nombrar el problema del prospecto, dar claridad sobre la solución, llamar a acción |
| Error común → explicación → acción | Señalar un error frecuente, explicar por qué pasa, indicar qué hacer |
| Duda frecuente → respuesta → cotizar | Formular la duda como el prospecto la diría, responder con autoridad, invitar a cotizar |
| Historia breve → aprendizaje → invitación | Contar un caso real o verosímil, extraer el aprendizaje, invitar a escribir |
| Documento → ruta posible → siguiente paso | Partir del documento que tiene la persona, mostrar qué ruta aplica, decir qué hacer |

## Formato: Caption

```
[HOOK — primera línea que detiene el scroll]

[CUERPO — 3 a 5 líneas con el mensaje central]

[CTA — acción específica a tomar]

[Hashtags — solo si se solicitan explícitamente]
```

**Ejemplo base:**
```
Si solo tienes cédula y quieres regresar a Venezuela, hay una ruta para ti.

No es la misma que la de quien tiene pasaporte. Pero existe, y se puede organizar con el acompañamiento correcto.

El primer paso es saber desde qué ciudad estás en EE.UU. y cuándo quieres viajar.

Escríbenos y evaluamos tu caso sin compromiso.
```

## Formato: Reel / Guión

```
[GANCHO — primeros 3 segundos, texto en pantalla + voz en off]

Escena 1:
- Visual: [descripción]
- Texto en pantalla: [texto exacto]
- Voz en off: [texto exacto]

Escena 2:
- Visual: [descripción]
- Texto en pantalla: [texto exacto]
- Voz en off: [texto exacto]

[Escenas adicionales según necesidad]

[CTA FINAL — texto en pantalla + voz en off]
```

**Ejemplo base:**
```
GANCHO: "¿Tienes pasaporte vencido y quieres regresar a Venezuela?"

Escena 1:
- Visual: mapa de EE.UU. con línea hacia Colombia
- Texto: "Pasaporte vencido no significa que no puedes viajar"
- Voz: "Si tu pasaporte está vencido, todavía tienes opciones reales"

Escena 2:
- Visual: ruta visual Miami → Táchira
- Texto: "Ruta por Colombia disponible para pasaporte vencido"
- Voz: "Una de las rutas que manejamos va por Colombia, con asistencia en el cruce"

CTA:
- Texto: "Escríbenos con tu ciudad en EE.UU. y tu fecha tentativa"
- Voz: "Cuéntanos desde qué ciudad estás y cuándo quieres viajar. Te orientamos."
```

## Formato: Carrusel

```
PORTADA:
- Titular impactante (pregunta o afirmación)
- Subtítulo opcional

SLIDE 2 a N:
- Slide [número]: [texto exacto del slide]

ÚLTIMO SLIDE:
- CTA claro
- Instrucción de acción

CAPTION SUGERIDO:
- Hook
- Resumen del carrusel
- CTA
```

**Ejemplo base (7 slides):**
```
PORTADA: "Tu documento cambia todo. ¿Sabes qué ruta te aplica?"

Slide 2: "No todos los venezolanos en EE.UU. pueden tomar la misma ruta"
Slide 3: "Si tienes pasaporte vigente → más opciones disponibles"
Slide 4: "Si tienes pasaporte vencido → hay rutas específicas para ti"
Slide 5: "Si solo tienes cédula → también existe una ruta, pero requiere salvoconducto"
Slide 6: "El error más común: asumir que lo que le funcionó a otro te funciona a ti"
Slide 7 (CTA): "Dinos qué documento tienes y desde qué ciudad estás. Te armamos la ruta."

CAPTION:
¿Cuántas personas creen que no pueden regresar porque no tienen la documentación correcta?

En XPLORATRIP evaluamos tu caso según tu documento real.

Escríbenos y encontramos la mejor ruta para ti. 👇
```

## Formato: Mensaje de WhatsApp

```
[Saludo directo y humano]

[Contexto breve de por qué se escribe]

[Mensaje central — claro y sin rodeos]

[CTA — acción específica]

[Pregunta de cierre — ciudad en EE.UU. + fecha tentativa cuando aplica]
```

**Ejemplo base — seguimiento a lead:**
```
Hola, ¿cómo estás?

Quedé pendiente de tu consulta sobre el regreso a Venezuela.

Quería comentarte que según el documento que tengas, hay rutas distintas disponibles para organizar tu viaje.

¿Me puedes confirmar desde qué ciudad en EE.UU. estarías saliendo y cuándo tienes pensado viajar?

Con esos datos te armo una orientación concreta para tu caso.
```

## Formato: Ideas de contenido

Cada idea debe incluir:

```
Objetivo: [atraer / educar / generar confianza / activar / convertir]
Formato: [reel / carrusel / caption / historia / flyer / WhatsApp]
Hook: [primera línea o imagen que detiene el scroll]
Mensaje central: [qué debe entender o sentir la persona]
CTA: [acción exacta que se espera]
```

**Ejemplo:**
```
Objetivo: educar + activar conversación
Formato: carrusel
Hook: "¿Solo tienes cédula venezolana y quieres volver a Venezuela desde EE.UU.?"
Mensaje central: existe una ruta específica para este caso, pero requiere salvoconducto y acompañamiento correcto
CTA: Escríbenos con tu ciudad actual y tu fecha tentativa para evaluar tu caso
```

## Checklist interno antes de publicar cualquier pieza

Antes de entregar o publicar, responder internamente:

1. ¿A qué tipo de prospecto le hablo?
2. ¿Cuál es la duda o dolor principal que resuelve esta pieza?
3. ¿El documento del viajero está contemplado en el mensaje?
4. ¿Estoy educando, conectando o convirtiendo?
5. ¿Cuál es la acción exacta que quiero provocar?
6. ¿Pedí ciudad actual en EE.UU. y fecha tentativa cuando aplica?
7. ¿La pieza es clara, útil y comercial a la vez?
8. ¿Hay CTA explícito?
